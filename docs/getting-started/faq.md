# Frequently Asked Questions

## General

### What is MadeForAI?

MadeForAI is a comprehensive, interactive learning platform designed to be the definitive resource for understanding Large Language Models (LLMs), Small Language Models (SLMs), and the entire GenAI ecosystem in 2026.

### Is MadeForAI free?

Yes! All our learning materials, notebooks, and documentation are completely free and open source. You can access everything on GitHub and run notebooks for free on Google Colab.

### Do I need a powerful computer?

No! All notebooks can be run on Google Colab, which provides free GPU access. For local development, most notebooks will work on any modern computer with 8GB+ RAM.

### Which learning path should I choose?

| Your Background | Recommended Path |
|-----------------|-----------------|
| New to programming | Start with basic Python, then Understanding AI |
| Know Python basics | Understanding AI (4-6 weeks) |
| Developer/Software Engineer | Engineering AI (8-12 weeks) |
| ML/Data Science experience | Researching AI (10-16 weeks) |

## Technical

### What languages/frameworks are used?

- **Python** - Primary programming language
- **PyTorch** - Deep learning framework
- **Hugging Face Transformers** - Model library
- **Jupyter Notebooks** - Interactive learning environment

### How do I run the notebooks?

**Option 1 (Recommended):** Click the "Open in Colab" badge at the top of any notebook.

**Option 2:** Clone the repo and run locally:
```bash
git clone https://github.com/madeforai/madeforai.git
cd madeforai
jupyter lab
```

### Are the notebooks compatible with Kaggle?

Yes! Most notebooks can also be run on Kaggle Notebooks with minimal modifications.
